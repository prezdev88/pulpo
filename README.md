## 🐙 About Pulpo

Pulpo is an open-source, beautifully crafted Git client built on Electron. It strips away the bloat and complexity of traditional Git clients, offering a minimalist, dark-themed workspace that puts your code first. Effortlessly browse your commit history, view precise side-by-side file diffs, manage your local branches, and review logs transparently with absolute elegance.

## ✨ Features

- **🚀 Lightning Fast:** Built with vanilla web technologies, ensuring zero overhead and instant interactions.
- **🎨 Premium Dark UI:** A gorgeous, glassmorphism-inspired dark theme designed to reduce eye strain and look fantastic.
- **🔍 Side-by-Side Diffs:** Review code changes instantly with an integrated dual-pane view for added and removed lines.
- **🌳 Seamless Branch Management:** Quickly create, switch, and overview your local branches without typing a single command.
- **👁️ Transparent Git Logs:** A built-in Git Execution Log window so you always know exactly what commands are running under the hood.
- **⚡ Native Feel:** Smooth zooming, multi-tab repository support, and responsive layouts.

## 🛠️ Installation

To run Pulpo locally on your machine, follow these simple steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/pulpo.git
   cd pulpo
   ```

2. **Install dependencies:**
   Make sure you have [Node.js](https://nodejs.org/) installed, then run:
   ```bash
   npm install
   ```

3. **Start the application:**
   ```bash
   npm start
   ```

## 💻 Technologies

Pulpo is proudly built with a lean, modern stack:
* **[Electron](https://www.electronjs.org/)** - For robust desktop integration and multi-process architecture.
* **Vanilla JavaScript (ES6+)** - For pure, high-performance logic without heavy framework payloads.
* **HTML5 & CSS3** - Custom CSS variables and Flexbox for a responsive, fluid design.
* **Git** - Direct, transparent interactions via standard local Git binaries.
